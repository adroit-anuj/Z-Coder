# <span style="color: #3498db;">Z-Coder</span>

<p>Welcome to <strong>Z-Coder</strong>, the ultimate platform for bookmarking and managing your unsolved or favorite coding problems! With Z-Coder, you can efficiently keep track of the problems you want to tackle and enjoy the convenience of our built-in compiler. We hope you have a fantastic experience!</p>

## <span style="color: #2ecc71;">Getting Started</span>

### <span style="color: #e74c3c;">How to Use the Website</span>

<ol>
    <li><strong>Fork and Download</strong>: Start by forking and downloading the Z-Coder repository.</li>
    <li><strong>Prerequisites</strong>: Ensure you have NodeJS and PostgreSQL installed on your system.</li>
    <li><strong>Install Dependencies</strong>: Run <code>npm init -y</code> to install the required modules.</li>
    <li><strong>Start the Server</strong>: Execute the command <code>node index.js</code>.</li>
    <li><strong>Access the Website</strong>: Open your browser and navigate to <code>localhost:3000/</code>.</li>
</ol>

<p><em>Note: Don't forget to unzip the vendor file located in <code>/public</code>.</em></p>

## <span style="color: #2ecc71;">Installation and Setup</span>

<p>To get started with Z-Coder on your local machine, follow these steps:</p>

<ol>
    <li><strong>Clone the Repository</strong>: Clone the Z-Coder repository to your local machine.</li>
    <li><strong>Install NodeJS and PostgreSQL</strong>: Ensure you have NodeJS and PostgreSQL installed.</li>
    <li><strong>Set Up Environment</strong>: Create an environment file and set up your PostgreSQL credentials.</li>
    <li><strong>Install Dependencies</strong>: Run <code>npm install</code> to install all necessary dependencies.</li>
    <li><strong>Database Migration</strong>: Run the necessary database migrations to set up your PostgreSQL database.</li>
    <li><strong>Start the Application</strong>: Use <code>node index.js</code> to start the server.</li>
    <li><strong>Access the Application</strong>: Open <code>localhost:3000</code> in your web browser to start using Z-Coder.</li>
</ol>
